# Инструкция по настройке Google Sheets

## 1. Создание таблицы и скрипта

1. Откройте [Google Sheets](https://sheets.google.com) → «Создать таблицу»
2. **Расширения** → **Apps Script**
3. Удалите код по умолчанию, вставьте содержимое файла `Code.gs`
4. Нажмите **+** → **Скрипт** → назовите `Analytics`
5. Вставьте содержимое файла `Analytics.gs`
6. Сохраните (Ctrl+S)

## 2. Публикация

1. **Развертывание** → **Новое развертывание**
2. Тип: **Веб-приложение**
3. Выполнять как: **Я**
4. Доступ: **Все** (Anyone)
5. Нажмите **Развернуть**
6. Скопируйте URL и вставьте в `script.js` вместо `YOUR_GOOGLE_APPS_SCRIPT_URL_HERE`

## 3. Аналитика

1. Обновите страницу таблицы
2. В меню появится **📊 Аналитика** → **Создать полный отчет**
3. Графики обновляются автоматически при каждом новом ответе
